# ヘボン式ローマ字変換ツール 実装指示

以下の仕様に基づき、日本語（ひらがな・カタカナ・半角カナ）をヘボン式ローマ字に変換するWebツールを実装してください。
画面に表示するメッセージ・ラベル・ボタン名等はすべて日本語にしてください。

## 1. 概要

- テキストボックスに入力した日本語をヘボン式ローマ字に変換して、別のテキストボックスに出力する
- 対応：ひらがな、カタカナ、半角カナ（正規化して変換）
- 漢字は変換せず、そのまま出力する
- 既存のローマ字（半角英字）が含まれる場合は、ヘボン式の表記ルールに正規化する
- それ以外（数字・記号・絵文字など変換対象外の文字）はそのまま出力する

## 2. 画面上部の説明文・注意文

タイトル直下に以下の説明文を表示する。

- 機能説明：「ひらがな・カタカナ・半角カナをヘボン式ローマ字に変換します。漢字はそのまま出力されます。」
- 自動変換とボタン変換の違いの説明：「入力中は自動で変換されます（単語の区切りなし）。『変換』ボタンを押すと、形態素解析を行って単語ごとに区切った結果に変換し直します。」
- 注意文：「変換結果は誤りを含む場合があります。出力内容は必ずご自身でご確認ください。」

## 3. ヘボン式変換ルール（設定パネル）

### プリセット（一括変更）

以下のプリセットを選択肢として用意し、選択すると対応する個別設定値を一括変更する。

| プリセット | 長音の表記 | 撥音「ん」 | 撥音＋母音・y続く場合の区切り |
|---|---|---|---|
| パスポート式（デフォルト） | 表記しない | b/m/pの前はm、それ以外はn | 何もしない |
| 鉄道式 | マクロン（ā, ī, ū, ē, ō） | b/m/pの前はm、それ以外はn | ハイフン（n-） |
| 道路標識式 | 表記しない | 常にn | 何もしない |
| カスタム | （個別設定の現在値に従う） | | |

- 個別設定項目を後から変更した場合、プリセットの表示は自動的に「カスタム」に切り替わる
- 個別設定の選択肢には、上記プリセット名（パスポート式等）は表示しない

### 個別設定項目

| 設定項目 | 選択肢（デフォルトは先頭） |
|---|---|
| 長音の表記 | 表記しない / マクロン（ā, ī, ū, ē, ō） / 母音を重ねる（例：おう→ou） |
| 撥音「ん」 | b/m/pの前はm、それ以外はn / 常にn |
| 撥音＋母音・y続く場合の区切り | 何もしない / アポストロフィ（n'） / ハイフン（n-） |
| 出力の文字幅 | 半角 / 全角 |
| アルファベットの大文字・小文字 | 小文字 / 大文字 / PascalCase |

### 変換例の表示

設定パネル内に、固定のサンプル文に対する変換結果を表示する。

- サンプル文：「しんぶんはきんようびにとうきょうへいく」
- 上記5つの個別設定（長音表記・撥音・区切り・出力文字幅・大文字小文字）のいずれかを変更すると、変換例の表示をリアルタイムに更新する
- 表示形式：「しんぶんはきんようびにとうきょうへいく → （変換結果）」

### PascalCaseのスペース設定

- 自動変換結果：常にスペースなし（例：`WatashiWaNihonJin`）
- 変換ボタン押下結果（形態素解析あり）：スペースあり／なしを選択可能（例：`Watashi Wa Nihon Jin` / `WatashiWaNihonJin`）

## 4. 半角カナの正規化

半角カナの濁点・半濁点（`ﾞ`/`ﾟ`）が基字と分離した2文字（例：`ｶ`+`ﾞ`）で入力された場合、対応する全角1文字（例：`ガ`）に正規化してから変換すること。

## 5. 単語区切り・大文字小文字の挙動

- **自動変換**（後述のリアルタイム変換）：単語区切りの判定は行わない。入力の文字位置・改行・スペースをそのまま維持して各文字を変換する
- **変換ボタン押下**：形態素解析ライブラリを使って単語区切りを判定し、単語間にスペースを挿入して出力する

## 6. 入力欄の仕様

- 複数行・改行に対応
- 文字数のハード上限はなし
- 文字数カウンターを表示する
- 一定文字数（目安：10,000文字）を超えた場合
  - 警告メッセージを表示する
  - 自動変換（リアルタイム変換）を停止する（変換ボタンでの変換は可能）
- 入力欄をクリアするボタンを設ける

## 7. 出力欄の仕様

- 読み取り専用（編集不可）
- 専用の「コピー」ボタンを設置し、クリップボードにコピーする
- コピー成功時はフィードバック（例：「コピーしました」のトースト表示）を行う

## 8. 変換トリガー

- 半角文字の入力時：デバウンス（一定時間入力が止まったら）で自動変換を実行する
- 全角文字の入力時：IMEの確定イベント（`compositionend`）のタイミングで自動変換を実行する
- 「変換」ボタン押下時：形態素解析を用いた全文の再変換を実行する
  - 上記の自動変換がうまく機能しなかった場合の保険としても機能する
  - ボタンの表示は2行で「変換」「（形態素解析）」とし、形態素解析を用いた変換であることを示す
- ヘボン式ルール設定（長音表記・撥音・大文字小文字等）を変更したとき、入力欄に文字が入っている場合は「設定が変更されました。変換ボタンを押して再変換してください」のようなメッセージを表示する

## 9. 形態素解析ライブラリ（変換ボタン用）

- ページ表示直後にバックグラウンドで形態素解析ライブラリ（辞書データ含む）の読み込みを開始する
- 読み込み中：「変換」ボタンは「準備中...」と表示し、無効化する
- 読み込み完了：ボタンのラベルを「変換」「（形態素解析）」に戻し、有効化する
- 読み込み失敗：エラーメッセージを表示し、「再試行」ボタンを表示する
- 自動変換（リアルタイム変換）はこのライブラリを使用しない（ライブラリの状態に関係なく常時動作する）

## 10. 変換不可文字の警告メッセージ

- 入力に変換できない文字（絵文字・特殊記号など）が含まれる場合、そのまま出力しつつ、出力欄付近に警告メッセージを表示する
- 変換実行（自動変換・ボタン変換いずれも）の都度、表示・非表示を更新する
- 該当文字が入力からなくなった時点でメッセージを消す

## 11. 設定の保持

- ブラウザのローカルストレージに以下を保存し、再読み込み後も復元する
  - プリセット選択状態、長音表記、撥音の表記、撥音＋母音/yの区切り、出力の文字幅、大文字・小文字（PascalCaseのスペース設定含む）
- 入力・出力テキストは保存しない

## 12. 画面レイアウト

### PC表示（上から順に配置）

1. タイトル
2. 説明文・注意文エリア（機能説明、自動変換とボタン変換の違い、注意文）
3. ヘボン式ルール設定パネル（プリセット、個別設定5項目、変換例の表示）
4. 入力欄／変換ボタン／出力欄を横並びに配置（左：入力欄＋文字数カウンター＋クリアボタン、中央：変換ボタン、右：出力欄＋コピーボタン）
5. 警告メッセージ表示エリア（変換不可文字がある場合のみ表示。それ以外は非表示）

### スマホ表示（上から順に縦並びで配置）

1. タイトル
2. 説明文・注意文エリア
3. ヘボン式ルール設定パネル（折りたたみ可能。デフォルトは折りたたみ状態）
4. 入力欄（高さを抑え、1画面に収まるようにする）＋文字数カウンター＋クリアボタン
5. 変換ボタン
6. 出力欄（高さを抑える）＋コピーボタン
7. 警告メッセージ表示エリア（変換不可文字がある場合のみ表示）

## 13. その他

- 変換処理中（特に変換ボタン押下時）はローディング表示を行う

## 画面表示メッセージ例（日本語）

- ボタン：「変換」「（形態素解析）」「準備中...」「再試行」「コピー」「クリア」
- フィードバック：「コピーしました」
- 警告：「変換できない文字が含まれています」「文字数が多いため、自動変換を停止しました」「設定が変更されました。変換ボタンを押して再変換してください」「形態素解析の読み込みに失敗しました」

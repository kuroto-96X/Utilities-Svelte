# NISA運用年率計算ツール（簡易版）実装指示

## 概要

NISA口座の取引画面に表示される「現在の評価額」「損益率」から、実際の運用年率（年率換算リターン）を逆算するWebツール。
NISAの取引画面では損益額・損益率は表示されるが、年率（年利）は表示されないため、それを補うツール。

積立投資（毎月/毎年の定額投資）を前提とし、Modified Dietz法による近似計算を使用する。
反復計算（ニュートン法等）は不要で、すべて閉じた式（四則演算・累乗）で計算できる。

フロントエンドのみで完結する想定（サーバーサイド計算不要）。モバイル表示を優先したレイアウトとする。

---

## 入力項目

| 項目 | 型 | デフォルト値 | 説明 |
|---|---|---|---|
| 投資開始年月 | `month`入力 | 任意（例: 2022-01） | 積立を開始した年月 |
| 投資頻度 | トグルボタン（毎月 / 毎年） | 毎月 | 積立の頻度 |
| 現在の評価額（円） | 数値入力 | 任意 | NISA画面に表示されている評価額 |
| 現在の損益率（%） | 数値入力（小数可、負数可） | 任意 | NISA画面に表示されている損益率 |
| 参考年率（%） | スライダー（0〜15、step 0.5） | 6.0 | 将来予測の比較用シナリオ年率。ユーザーが自由に変更可能 |

---

## 計算ロジック

### 1. 投資期数 N の算出

現在年月と投資開始年月の差分から算出する。

- 毎月の場合: `N = (現在の年 - 開始年) * 12 + (現在の月 - 開始月)`
- 毎年の場合: `N = 現在の年 - 開始年`

`N < 1` の場合は `N = 1` に補正する（投資開始直後で期数がゼロ・負になる場合の防止）。

### 2. 累計投資額・1回あたり投資額

```
累計投資額 = 現在の評価額 / (1 + 損益率/100)
1回あたり投資額 = 累計投資額 / N
```

### 3. Modified Dietz法による期間全体のリターン R

定額・定期投資を前提に、各回の投資が平均して期間の約半分運用されてきたとみなして近似する。

```
R = (損益率/100) * (2N / (N+1))
```

### 4. 年率換算リターン r

```
T = N / (毎月なら12, 毎年なら1)   ← 年数
r = (1 + R)^(1/T) - 1
```

### 5. 将来価値の計算（将来予測テーブル用）

年率 `rateAnnual`（自分の年率 r、または参考年率）で、今後 `years` 年間、同じペース・同じ1回あたり投資額で投資を継続した場合の評価額。

```
periodsPerYear = 毎月なら12, 毎年なら1
M = years * periodsPerYear
rPeriod = (1 + rateAnnual)^(1/periodsPerYear) - 1

将来評価額 = 現在の評価額 * (1 + rPeriod)^M
           + 1回あたり投資額 * (((1 + rPeriod)^M - 1) / rPeriod)
```

`rPeriod` が0に近い場合（年率0%）は、上式の第2項が `0/0` になるため、以下に置き換える。

```
将来評価額 = 現在の評価額 + 1回あたり投資額 * M
```

### 6. NISA生涯投資枠（1,800万円）満額到達予測

```
NISA_MAX = 18,000,000円
残り投資可能額 = NISA_MAX - 累計投資額

if 残り投資可能額 <= 0:
    満額到達済み（追加投資なし、既存資産の成長のみ表示）
else:
    満額到達までの期数 = 残り投資可能額 / 1回あたり投資額
    満額到達までの年数 = 満額到達までの期数 / periodsPerYear
    満額到達時の将来評価額 = 上記5の式に years=満額到達までの年数 を代入
```

## 出力項目

### 計算結果（メトリックカード、2x2グリッド）

- 年率換算リターン（%、小数1桁）
- 運用期間（年、小数1桁）
- 累計投資額（円、3桁区切り）
- 1回あたり投資額（円、3桁区切り）

### 将来予測テーブル（単位: 万円）

| 行 | ご自身の年率での予測 | 参考年率での予測 |
|---|---|---|
| 5年後 | 上記5の式（years=5, rateAnnual=r） | 上記5の式（years=5, rateAnnual=参考年率） |
| 10年後 | 同上（years=10） | 同上 |
| 20年後 | 同上（years=20） | 同上 |
| NISA満額到達時（または「満額達成済み」） | 上記6の式 | 同上 |

表示は万円単位に変換（`Math.round(円 / 10000)`）。

## 画面レイアウト（上から順）

1. **入力カード**
   - 投資開始年月（month input）
   - 投資頻度トグル（毎月 / 毎年、選択状態を強調表示）
   - 現在の評価額（円） / 現在の損益率（%）を横並び2カラムで配置（この順序）

2. **計算結果カード（2x2グリッド）**
   - 年率換算リターン / 運用期間 / 累計投資額 / 1回あたり投資額

3. **将来予測カード**
   - 参考年率スライダー（0〜15%、step 0.5、デフォルト6%）。スライダー操作で右側ラベルとテーブルがリアルタイム更新される
   - 将来予測テーブル（5年後 / 10年後 / 20年後 / NISA満額到達時）
   - 免責文（下記参照）

すべての入力値変更・スライダー操作で、計算結果と将来予測テーブルをリアルタイム再計算・再表示する。

## エッジケース処理

- **N = 0（投資開始月 = 現在月）**: `T = 0` となり年率計算が不可能（ゼロ除算）。`N >= 1` に補正することで回避するが、補正している旨を画面下部に小さく注記してもよい。
- **損益率が -100% 以下（全損またはそれ以上）**: `累計投資額 = 評価額/(1+損益率/100)` が0または負になり計算が破綻する。この場合は計算結果を表示せず、「入力値を確認してください」というエラーメッセージを表示する。
- **累計投資額が既にNISA満額（1,800万円）を超えている**: 「満額到達」の行は「満額達成済み」と表示し、将来評価額は新規投資分（1回あたり投資額の項）を含めず、既存資産の成長のみで計算する（`years=0`扱い、または式の第2項を省略）。
- **満額到達までの年数が極端に長い（例: 50年超）**: 「現実的ではない長期間」として、参考値である旨の注記を強調する（必須ではないが推奨）。

## 免責・注記文（画面下部に小さく表示）

```
※年率は損益率と投資期間から近似計算した参考値です。投資額が一定でない場合や
不定期な投資の場合、実際の年率とは異なります。
※参考年率は想定値であり、将来の運用成果を示すものではありません。
※NISA生涯投資枠1,800万円（2024年からの新NISA制度）を基準にしています。
制度内容は変更される可能性があります。
```

## 技術メモ

- すべての計算はクライアントサイドJavaScriptで完結する（サーバー通信不要）。
- 画面に表示するすべての数値は `Math.round()` または `toFixed()` で丸め、`toLocaleString()` で3桁区切り表示する。
- モバイル幅（〜400px程度）を基準にレイアウトを組み、入力欄・カードは縦並びを基本とする。
- 将来予測テーブルは横スクロールが発生しないよう、列数を最小限（ラベル列＋2列）に抑える。
